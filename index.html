Install both Druckertreiber (DDST Printer driver) and Dienstprogramm (Smart Organising Monitor) in the same order from the official Ricoh website.

How to troubleshoot if DDST Printer driver installation fails (First YouTube video shows the process of installation clearly)

  • Open Settings
  • Printers
  • Open Settings
  • Add Printers
  • Printer that I want isn't listed
  • My Printer is a little older. Help me find it.
  • Click next
  • Under list of devices select Ricoh SP112 printer
  • Have a disk
  • Select the unpacked DDST driver from the specific folder

Another way of installation

  • Open Device Manager from windows search menu
  • Scan for Hardware changes
  • Select the usb device with yellow caution symbol ⚠️
  • Update driver
  • Select the folder and continue like previous step
